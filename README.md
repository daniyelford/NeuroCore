# NeuroCore

NeuroCore is a high-performance machine learning engine written in Go.

## Goals

- Tensor Engine
- Autograd
- Deep Learning
- Bioinformatics
- High Performance Computing

## Status

🚧 At the end of version 1 and Under Development for next version

## Install

`bash
go run ./cmd/NeuroCore "examples/json_classifier/data.json" 50
`bash
